# Personal Portfolio Website

A modern, responsive personal portfolio built using **Spring Boot, Thymeleaf, HTML, and CSS**.  
This project showcases my academic journey, technical skills, and personal growth as a BSIT student.

---

## Overview

This portfolio website serves as a personal digital space to present my profile as an IT student, including my background, experiences, skills, and student life journey.

It is designed with a clean and minimalist UI/UX approach, focusing on simplicity, structure, and aesthetic presentation.

---

## Features

- Home Page – Personal introduction with modern layout  
- About Page – Biography, interests, and goals  
- Journey Page – Timeline of my college experiences  
- Skills Page – Programming skills with progress bars  
- Gallery Page – Student life moments with captions  
- Clean and aesthetic UI design  
- Smooth hover effects and modern interface  

---
